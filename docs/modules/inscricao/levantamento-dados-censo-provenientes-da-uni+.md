# Levantamento de Dados Informados no Censo da Educação Superior Provenientes do Sistema Uni+

Anualmente, as Instituições de Ensino Superior (IES) devem informar ao Censo da Educação Superior, por meio do sistema CENSUP, os dados acadêmicos e cadastrais de seus alunos.

Considerando essa obrigação, o sistema Uni+ deve manter compatibilidade com os formatos, tipos de dados e regras de preenchimento dos campos sob sua responsabilidade, que são utilizados na geração do arquivo de migração enviado ao CENSUP.

Com o objetivo de apoiar esse processo e identificar os dados que podem ser obtidos diretamente do sistema Uni+, foi realizado o presente levantamento com base no dicionário de dados disponibilizado pelo CENSUP para o Censo da Educação Superior de 2026, referente às informações do ano-base 2025.

A Tabela 1 apresenta os campos que podem estar armazenados no sistema Uni+ e que devem ser considerados na composição do arquivo de migração exigido pelo CENSUP.
